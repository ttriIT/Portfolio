Async/Await trong JS cho I/O mạng

Ngày đăng: 28/10/2023 Tác giả: Nguyễn Văn A

Xử lý lỗi với try/catch

async function fetchJson(url) {
  try {
    const res = await fetch(url);
    if (!res.ok) throw new Error(`HTTP ${res.status}`);
    return res.json();
  } catch (err) {
    console.error('Request failed:', err);
    throw err;
  }
}

Hủy request bằng AbortController

const controller = new AbortController();
const id = setTimeout(() => controller.abort(), 5000);

fetch('/api/data', { signal: controller.signal })
  .then(...)
  .catch(err => {
    if (err.name === 'AbortError') console.log('Aborted');
  });

Batch song song an toàn

const urls = ['a.json', 'b.json', 'c.json'];
const results = await Promise.allSettled(urls.map(fetchJson));
const ok = results.filter(r => r.status === 'fulfilled').map(r => r.value);

Timeout và retry backoff

async function withRetry(fn, { retries = 3, base = 500 } = {}) {
  for (let i = 0; i <= retries; i++) {
    try { return await fn(); } catch (e) {}
    const delay = Math.min(30000, base * 2 ** i);
    await new Promise(r => setTimeout(r, delay));
  }
  throw new Error('Max retries');
}

Kết luận

Async/Await giúp code gọn gàng, nhưng cần quản lý lỗi, timeout, hủy request và batch cẩn thận để hệ thống ổn định.

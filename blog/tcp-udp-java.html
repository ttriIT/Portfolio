Tổng quan TCP/UDP trong Java

Ngày đăng: 24/10/2025 Tác giả: Nguyễn Thiện Trí

Giới thiệu về TCP và UDP

TCP (Transmission Control Protocol) và UDP (User Datagram Protocol) là hai giao thức truyền tải dữ liệu phổ biến nhất trong mô hình TCP/IP. Mỗi giao thức có những đặc điểm và ứng dụng riêng biệt.

TCP trong Java

TCP là giao thức hướng kết nối, đảm bảo độ tin cậy cao trong việc truyền dữ liệu. Trong Java, lập trình TCP được thực hiện thông qua các lớp Socket và ServerSocket.


// Ví dụ TCP Server
ServerSocket serverSocket = new ServerSocket(8080);
Socket clientSocket = serverSocket.accept();
BufferedReader in = new BufferedReader(new InputStreamReader(clientSocket.getInputStream()));
PrintWriter out = new PrintWriter(clientSocket.getOutputStream(), true);

// Ví dụ TCP Client
Socket socket = new Socket("localhost", 8080);
BufferedReader in = new BufferedReader(new InputStreamReader(socket.getInputStream()));
PrintWriter out = new PrintWriter(socket.getOutputStream(), true);

UDP trong Java

UDP là giao thức không kết nối, ưu tiên tốc độ hơn độ tin cậy. Trong Java, lập trình UDP được thực hiện thông qua lớp DatagramSocket và DatagramPacket.


// Ví dụ UDP Server
DatagramSocket socket = new DatagramSocket(8080);
byte[] buffer = new byte[1024];
DatagramPacket packet = new DatagramPacket(buffer, buffer.length);
socket.receive(packet);

// Ví dụ UDP Client
DatagramSocket socket = new DatagramSocket();
InetAddress address = InetAddress.getByName("localhost");
byte[] buffer = "Hello, UDP!".getBytes();
DatagramPacket packet = new DatagramPacket(buffer, buffer.length, address, 8080);
socket.send(packet);

So sánh TCP và UDP

Đặc điểm TCP UDP
Kết nối Hướng kết nối Không kết nối
Độ tin cậy Cao (có cơ chế xác nhận) Thấp (không có cơ chế xác nhận)
Tốc độ Chậm hơn Nhanh hơn
Thứ tự gói tin Đảm bảo Không đảm bảo
Ứng dụng Web, Email, FTP Video streaming, VoIP, Game online

Khi nào sử dụng TCP hoặc UDP?

Việc lựa chọn giữa TCP và UDP phụ thuộc vào yêu cầu cụ thể của ứng dụng:

  • Sử dụng TCP khi: Cần đảm bảo dữ liệu được truyền đầy đủ và chính xác (như truyền file, web browsing, email).
  • Sử dụng UDP khi: Cần tốc độ cao và có thể chấp nhận mất một số gói tin (như video streaming, game online, VoIP).

Kết luận

Hiểu rõ về TCP và UDP trong Java là nền tảng quan trọng cho việc phát triển các ứng dụng mạng. Mỗi giao thức có những ưu điểm và hạn chế riêng, việc lựa chọn giao thức phù hợp sẽ giúp tối ưu hiệu suất của ứng dụng.

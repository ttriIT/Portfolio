Streaming dữ liệu với Node.js

Ngày đăng: 29/10/2025 Tác giả: Nguyễn Thiện Trí

Tải file hiệu quả

// Stream file đến client
app.get('/download', (req, res) => {
  const stream = fs.createReadStream('bigfile.zip');
  stream.pipe(res);
});

Proxy dữ liệu và backpressure

http.request(opts, (upstream) => {
  upstream.on('data', chunk => {
    if (!client.write(chunk)) upstream.pause();
  });
  client.on('drain', () => upstream.resume());
});

Pipeline và transform

const { pipeline } = require('stream');
pipeline(source, transform, destination, (err) => {
  if (err) console.error('Pipeline failed', err);
});

Kết luận

Stream phù hợp cho dữ liệu lớn, giảm bộ nhớ và tăng tốc độ. Luôn xử lý backpressure để tránh nghẽn và mất dữ liệu.

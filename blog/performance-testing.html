Kiểm thử hiệu năng: JMH và k6

Ngày đăng: 01/11/2025 Tác giả: Nguyễn Thiện Trí

Micro-benchmark với JMH

@Benchmark
public void measure() {
  // Đo đoạn code cụ thể
}

Thiết lập warmup, measurement, forks để kết quả ổn định; theo dõi throughput và phân phối latency.

Kiểm thử tải API với k6

import http from 'k6/http';
import { sleep } from 'k6';

export default function() {
  http.get('https://api.example.com/v1/items');
  sleep(1);
}

Đặt VUs, duration, stages; theo dõi error rate, p95/p99 latency và xử lý bottlenecks dựa trên metrics.

Kết luận

Xây dựng baseline, đo lường liên tục và tối ưu dựa trên profile thực tế để cải thiện hiệu năng.
